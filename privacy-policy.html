2. Information We Collect

We collect information that you voluntarily provide when using the application:

3. Data Storage

User data is securely stored and managed using Supabase, which provides backend infrastructure and database services. Supabase follows industry-standard security practices to protect user data.

4. How We Use Your Information

We use collected information to:

  • Create and manage user accounts.

  • Enable community chat features.
  • Process subscriptions and premium services.
  • Improve application performance and user experience.
  • Communicate service updates or important notices.

5. Third-Party Services

Habit Hub may use third-party services that collect information automatically to improve functionality:

6. Data Retention & Deletion

We retain personal data only as long as necessary to provide our services. Users can delete their account directly from the **Settings** section of the app. Once an account is deleted, associated personal information is permanently removed from our systems.

7. Children's Privacy

Habit Hub is intended for users aged 13 years and older. We do not knowingly collect personal information from children under the age of 13.

8. Data Security

We implement reasonable technical and organizational security measures to protect user information from unauthorized access or disclosure.
